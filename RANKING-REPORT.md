## Technical SEO Audit

### Critical Issues (Fix Immediately)
| Issue | Impact | Details |
|-------|--------|---------|
| **No sitemap.xml** | HIGH | Google can't efficiently crawl the site |
| **No robots.txt** | HIGH | No crawl directives for search engines |
| **No structured data/schema** | HIGH | Zero JSON-LD markup — no LocalBusiness, Service, FAQ, Review schema |
| **Mobirise platform** | HIGH | Limited SEO control, no CMS for blog/content updates |
| **No blog** | HIGH | Zero content marketing — no way to rank for informational queries |
| **HTTP mixed content** | MEDIUM | Some internal links use http:// instead of https:// |
| **Typos in URLs** | MEDIUM | `french_drains_oakland_michgian.html` (misspelled "Michigan") |
| **No canonical tags** | MEDIUM | Potential duplicate content issues |

### Major Issues
| Issue | Impact | Details |
|-------|--------|---------|
| **No Google Business Profile link** | HIGH | Missing critical local SEO signal |
| **No meta descriptions on subpages** | HIGH | Google auto-generates snippets (often poorly) |
| **Thin service pages** | MEDIUM | Drainage page ~600 words; lighting, outdoor living even thinner |
| **No FAQ schema** | HIGH | Missing AEO opportunity — can't appear in AI answers |
| **No appointment/booking tool** | HIGH | Contact form only — no modern scheduling |
| **25+ testimonials not in schema** | HIGH | Reviews aren't visible to Google as structured data |
| **Images not optimized** | MEDIUM | .gif format used, no WebP, questionable compression |
| **No Google Analytics 4 goals** | MEDIUM | GA tag present but no conversion tracking visible |

## Recommendations Summary (Prioritized)

### Phase 1: New Website Build (Immediate)
1. Rebuild on modern platform (React/Next.js or high-quality HTML/CSS)
2. Add comprehensive structured data (LocalBusiness, Service, FAQ, Review, VideoObject schemas)
3. Create 1,500+ word service pages for each core service
4. Build FAQ sections for every service with FAQ schema
5. Embed YouTube videos throughout relevant service pages
6. Integrate appointment scheduling (Calendly or similar)
7. Add sitemap.xml, robots.txt, canonical tags
8. Fix all spelling errors in URLs and content
9. Add Google Business Profile widget/link
10. Implement before/after gallery with project descriptions

### Phase 2: Content Strategy (Weeks 1-4)
1. Launch blog with 2-3 posts per week targeting informational keywords
2. Create "Ultimate Guide" pages for each service area
3. Add video content (longer educational videos)
4. Build FAQ database (50+ questions across all services)
5. Write detailed project case studies

### Phase 3: Social & Local SEO (Ongoing)
1. Set up TikTok, Instagram (business), X accounts
2. Optimize Google Business Profile
3. Build local citations (Yelp, BBB, HomeAdvisor, Angi)
4. Implement review generation system
5. Cross-post video content across platforms
